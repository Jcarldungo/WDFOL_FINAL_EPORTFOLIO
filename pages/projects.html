SnapMark — Public Bookmarking App

A full-stack bookmarking app demonstrating a complete Node.js → MongoDB Atlas → Netlify Functions deployment. Users save links with titles, descriptions, and tags. Data persists to a real cloud database — this is a genuine REST API with GET, POST, and DELETE endpoints connected to MongoDB Atlas.

  • Real MongoDB Atlas database — data persists across all users and sessions
  • Three REST API endpoints: GET, POST, DELETE via Netlify serverless functions
  • Input validation on both server (Node.js) and client (JavaScript)
  • Tag-based filtering, Google Favicon API integration, live API response preview panel
